# runPacer
Bash script containing sox commands to add a 180BPM beat to soundfiles 

# requires
SoX with mp3 extension and id3v2 (or comment out lines 44 and 45)

# set permissions and make executable
chmod u+x runPacer.sh

# to run
./runPacer.sh
